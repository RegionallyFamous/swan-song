# Swan Song homebrew lab

This directory contains original WonderSwan software and reusable creation
workflows developed alongside Swan Song. It does not contain commercial ROMs,
Bandai BIOS files, WonderWitch firmware, or copied game assets.

## SWANFRAME

[SWANFRAME](swanframe/README.md) is a self-contained WonderSwan Color music
player with five original four-channel chiptunes, an original SD-mecha mascot,
and the native Night Drive interface. The GPLv3 project includes reproducible
source-art conversion, a 128 KiB ROM recipe, per-track SwanSong/ares capture
plans, audio verification, and a deterministic source-inclusive package.

Physical WonderSwan-family, flash-cartridge, Analogue Pocket, and Dock testing
remain pending. Treat the checked-in evidence as emulator evidence only.

## Reusable skill

[Make WonderSwan Music](skills/make-wonderswan-music/SKILL.md) documents the
console's four wavetable channels, sweep and noise features, composition and
audience workflow, SWANFRAME score format, linting, catalog auditing, and
release-proof sequence. The repository copy matches the installed Codex skill.
