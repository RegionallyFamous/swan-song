# SWANFRAME UI colorway prompt — Night Drive v2

Mode: OpenAI ImageGen built-in tool, reference-guided UI colorway edit.

References: the approved SWANFRAME v1 cockpit-player concept and original v5
mascot identity art. Composition and character silhouette were locked; only the
product color system changed.

## Colorway

- Midnight violet `#08061c`: deepest background.
- Smoky indigo `#292451`: frames, inactive rails, inner robot structure.
- Hot orchid `#f04aa6`: armor, track numerals, selected states.
- Phosphor mint `#65f3c3`: sensors, playback information, live meters.

The edit removed the former amber/orange/cyan scheme while preserving the left
mascot portrait, dominant track number, isolated title, segmented timeline,
four waveform lanes, and icon dock. The result remains flat, hard-edged,
tile-aware pixel art with no gradients or added interface prose. SWANFRAME stays
an original non-franchise mecha with its asymmetric tuning antenna; no V-fin,
licensed marks, extra characters, desktop chrome, or watermark.

The ROM reconstructs the interface natively and uses the generated image only
as color and layout art direction.
