## Prompt

PRECISION CHARACTER REDESIGN / REFERENCE-GUIDED EDIT.

Use the supplied SWANFRAME image as the identity reference. Create a cleaner v5
of the SAME original mascot for conversion into a 72×64 four-color WonderSwan
sprite.

Preserve these identity invariants:

- Original super-deformed real-robot music mascot named SWANFRAME; never depict
  an existing licensed mobile suit.
- About two heads tall: enormous mechanical helmet/head, tiny compact torso,
  stubby limbs, broad block feet.
- Burnt amber armor, dark gunmetal frame, bright cyan optical/music accents,
  very dark navy background.
- One asymmetric tuning-fork / antenna blade, a circular chest tuner core,
  compact radiator-like backpack shapes, and a right forearm mixer/controller
  gauntlet.
- Confident neutral front three-quarter stance, full body visible.

Targeted v5 improvements:

  cockpit, human pilot, scenery, floor, cast shadow, or extra props.
- Avoid a twin V-fin, red chin, white/blue/red hero palette, or any exact Gundam

Rendering contract:

- Square character-design image, full body centered, about 82% of canvas
  height, generous safe padding.
- True crisp pixel-art aesthetic with deliberate chunky clusters and hard

- Flat graphic color blocking only. Use exactly these visual colors with no
  gradients, bloom, texture, glow, atmospheric lighting, transparency, or

- Quiet solid background, strongest silhouette possible, suitable for
  downsampling and strict four-color quantization.
- Polished late-1980s handheld-game SD-mecha character portrait, but fully
