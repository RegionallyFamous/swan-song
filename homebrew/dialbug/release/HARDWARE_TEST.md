# Physical hardware test

Status: **pending**

This does not block the emulator-qualified 0.1.0 public release. It does block
claims that Dialbug is physically qualified for a WonderSwan-family console,
flash cartridge, Analogue Pocket, or Dock.

Before calling this a hardware release, test the exact ROM hash recorded in
`verification-report.json` on at least one WonderSwan Color or SwanCrystal and
a supported flash cartridge. Confirm boot, all five songs, headphones, built-in
speaker, pause/resume, volume, every speed, every channel mute, loop seams, LCD
readability, and a 30-minute soak. Record the console model, flash cartridge,
firmware, date, tester, and any audio/video evidence here.
