# License status

Dialbug was newly authored in the Swan Song work
tree, but the maintainer has not yet declared a license for it. The local ROM,
evidence, and source-inclusive package are therefore a development preview for
authorized testing, not a public distribution authorization.

Before publishing, the maintainer should choose and record a license for the
new source and original soundtrack, review the notices for statically linked
Wonderful Toolchain libraries, and confirm the attribution language in
`CREDITS.md`.

No Humming Cat source, binary, mascot artwork, name treatment, or song data is
included. Its project and archive are cited only because the original
documentation informed the low-level player/sequencer research.
